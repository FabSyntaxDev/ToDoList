⚡ Todoist - Gerenciador de Tarefas
O Todoist é uma aplicação web leve e intuitiva para o gerenciamento de tarefas diárias. O projeto foca em oferecer uma experiência de usuário fluida, permitindo organizar afazeres por prioridade e acompanhar o progresso em tempo real através de um painel de estatísticas dinâmico.

🚀 Funcionalidades
Gestão de Tarefas: Adição, conclusão e exclusão de itens de forma simples.

Níveis de Prioridade: Classificação de tarefas em Baixa, Média ou Alta, com estilização visual distinta para cada uma.

Persistência de Dados: Utiliza localStorage para garantir que suas tarefas permaneçam salvas mesmo após fechar o navegador.

Dashboard de Estatísticas: Visualização rápida do total de tarefas, tarefas ativas e concluídas.

Interface Responsiva: Design adaptável para diferentes tamanhos de tela, incluindo dispositivos móveis.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica da aplicação.

CSS3: Estilização moderna com uso de Flexbox, Grid e gradientes.

JavaScript (ES6+): Lógica de manipulação de estado, renderização dinâmica e armazenamento local.
